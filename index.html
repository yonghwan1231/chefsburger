포트폴리오 소개

제작기간 : 22.02.14 ~ 22.03.01 (15일)

제 이력서와 포트폴리오를 열람해 주셔서 진심으로 감사드립니다!
저의 첫 포트폴리오이며, 데스크탑 전용 웹사이트로 만들었습니다.
제가 수료한 학원은 여타 학원의 클론 코딩과는 다르게 직접 기획한 홈페이지로 포트폴리오를 제작하였습니다.
레이아웃, 이미지, 효과 등 모든 요소를 직접 구상하였고, 웹 표준을 준수하여 미완성된 페이지 없이 꼼꼼히 작업하였습니다.
header와 footer는 공통영역이기 때문에 하나의 css 파일로 만들어 모든 페이지와 연결 시켰습니다.
열심히 만든만큼 보시는 분들께도 저의 정성과 노력이 온전히 전해졌으면 좋겠습니다.
